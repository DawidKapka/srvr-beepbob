# srvr-beepbob
simple Discord Bot for srvr
### Run locally

- **create ```config.json```**
```typescript
{
  "token": "<discord_token>"
}
```
- **install ```node``` & ```npm```**

- **compile & run**
```
tsc -p .
node src/index.js
```
**or**
```
npm start
```

*
configured for specific discord server
to run on other server, adjust id's in ```src/resources/id.json```

might add a config file for adjusting later...
